The purpose of this assignment is to build a Single Page Application powered by React and TypeScript. This application should consist of at least one single page that will display a list of users.

On this page there should be a form that will allow to add, edit or delete users following CRUD principles. It is up to you to decide how the form should behave and where it should be placed.

In order to get data, you can use a free fake API such as {JSON} Placeholder, or you can feel free to use another fake API service.

  • The application should be written with React and TypeScript. You can use a boilerplate such as create-react-app.
  • The application should fetch data from an external API and display a list of users.
  • The application should allow users to be added, deleted or edited.
  • The application needs to follow a component-based structure.
  • The application should respect Algolia design guidelines as much as possible. You will find below a list of our most important design primitives.
  • Bonus points if you add your personal touch to the page (it can be CSS animations, custom components, new interactions... Just be creative!)

Colors

  • #000033 (Xenon-900): Dark background
  • #003DFF (Xenon-600): Accent color such as links or icons (light background)
  • #457AFF (Xenon-400): Accent color such as links or icons (dark background)

  • #484C7A (Grey-700): Body text (light background)

Typography

  • Sora: Headings and buttons
  • Inter: Body text
  • Please note that those fonts are available via Google fonts.

Spacings

Margins and paddings are based on multiples of 8. They are set in pixels.
